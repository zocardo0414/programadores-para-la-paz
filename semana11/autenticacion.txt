El login es el proceso mediante el cual un usuario se identifica en una aplicación utilizando sus credenciales, como un correo electrónico y una contraseña. La autenticación permite verificar la identidad del usuario antes de otorgarle acceso al sistema. Algunas rutas deben estar protegidas porque contienen información privada o permiten realizar acciones importantes que solo deben ejecutar usuarios autorizados. No deben exponerse públicamente contraseñas, documentos de identidad, direcciones, teléfonos, tokens, claves de API ni ningún otro dato sensible que pueda poner en riesgo la seguridad y la privacidad de los usuarios.
