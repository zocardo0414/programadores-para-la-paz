Backlog priorizado

1. Como ciudadano quiero registrar una incidencia para dejar evidencia del problema y facilitar su seguimiento.

2. Como coordinador quiero recibir una notificación en Telegram cuando se registre una nueva incidencia para revisarla rápidamente.

3. Como miembro del equipo quiero consultar el listado de reportes para hacer seguimiento a cada caso.

4. Como administrador quiero actualizar el estado de una incidencia para mantener informada a la comunidad.

5. Como ciudadano quiero conocer el estado de mi reporte para saber si ya fue atendido.
